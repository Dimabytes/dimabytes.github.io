# HTML templates of "Leave me"

## [demo](https://dimabytes.github.io/leave-me-template/)

I created this project before ["Leave me" on React](https://github.com/Dimabytes/leave-me) to practice in making templates.


## Base scripts

* ### Install

```
npm install
```

* ### Start

```
npm run start
```

* ### Build

```
npm run build
```

* ### Dev build

```
npm run dev
```


* ### Watch mode

```
npm run watch
```
